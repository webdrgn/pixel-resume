Не скучный PDF. Живой диалог с виртуальным интервьюером, который адаптируется под твои ответы.

## 🚀 Что это

5-минутная сессия в формате Zoom-звонка. Ты отвечаешь на вопросы, выбираешь стратегию поведения — система оценивает стиль коммуникации. В конце — детальный разбор с метриками и рекомендациями.

### Для кого

| Аудитория | Зачем |
|-----------|-------|
| **Кандидаты** | Прокачать навыки собеседований без риска |
| **HR и рекрутеры** | Оценить soft skills в игровой форме |
| **Компании** | Использовать как часть отбора или онбординга |

## 💡 Ключевые фичи

| | |
|---|---|
| 🎭 **Иммерсивный формат** | Интерфейс видеозвонка с аватаром интервьюера |
| 🧠 **Адаптивные вопросы** | Сложность подстраивается под твои ответы |
| 📊 **Аналитика** | Оценка уверенности, аргументации и стрессоустойчивости |
| 🌍 **Доступность** | Работает в браузере, без установок |
